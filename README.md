# FitLog

## Overview
The FitLog App is a full-stack web application that allows users to create and manage personalized workout routines. After signing up and logging in, users can build routines, add exercises, and track details such as sets, repetitions, and weight. Each routine can be set to public or private, allowing users to either keep their workouts personal or share them with others.

## User Stories
* As a user, I want to be able to create an account and securely log in, so I can access and manage my personal workout data.

* As a user, I want to create workout routines and give each routine a custom name, so I can organize my training plans.

* As a user, I want to add exercises to a routine, so I can build a complete workout.

* As a user, I want to edit or delete my routines and exercises, so I can keep my workouts up to date.

* As a user, I want the option to make my routines public or private, so I can choose whether to share them with others.

* As a user, I want to browse and view public routines created by other users, so I can get inspiration and discover new workouts.

* As a user, I want to log sets for each exercise, including weight and number of repetitions, so I can track my progress over time.

* As a user, I want to edit or delete my logged sets, so I can correct mistakes or update my performance.

* As a user, I want a clear and organized view of my workout history, so I can easily monitor my improvement.

* As a user, I want to mark an exercise as completed using a checkbox, so I can track which exercises I have finished in my workout.

## Routes
| **Method** |    **Route**    |       **Description**       |
|:----------:|:---------------:|:---------------------------:|
|     GET    |        /        |           Homepage          |
|     GET    |    /routines    |      List all routines      |
|     GET    |  /routines/new  |       New routine form      |
|    POST    |    /routines    |        Create routine       |
|     GET    |    /routines    | View routine with exercises |
|     GET    | /routines//edit |      Edit routine form      |
|     PUT    |    /routines/   |        Update routine       |
|   DELETE   |    /routines/   |        Delete routine       |


## Features
* User authentication (sign up, log in, and log out)
* Create and manage personalized workout routines
* Add exercises to each routine
* Track the number of sets for every exercise
* Record weight and repetitions for progress tracking
* Set routines as public or private to control visibility
